# Goliat Gatsby

Este proyecto fue migrado desde un archivo index.html a Gatsby.

## Scripts

- `npm run develop` — Inicia el servidor de desarrollo
- `npm run build` — Compila el sitio para producción
- `npm run serve` — Sirve el sitio compilado

## Estructura
- `src/pages/index.js` — Página principal
- `src/pages/404.js` — Página de error

## Requisitos
- Node.js >= 18

## Migración
El contenido original de `index.html` debe ser adaptado en `src/pages/index.js`.
